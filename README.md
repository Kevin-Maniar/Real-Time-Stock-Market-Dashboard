# \# 📊 Equity Performance Monitor

# 

# A real-time stock market dashboard built using Streamlit to track and analyze stock performance with an interactive and simple UI.

# 

# ---

# 

# \## 🚀 Features

# 

# \* 📈 Real-time stock price tracking

# \* 📊 Interactive charts using Plotly

# \* 🔄 Auto-refresh for live updates

# \* 📉 High, Low, Open price display

# \* 📦 Clean and minimal dashboard UI

# 

# ---

# 

# \## 🛠️ Tech Stack

# 

# \* Python 🐍

# \* Streamlit

# \* Pandas

# \* yFinance

# \* Plotly

# 

# ---

#

# \## ⚙️ Installation \& Setup

# 

# 1\. Clone the repository:

# 

# ```

# git clone https://github.com/your-username/your-repo-name.git

# ```

# 

# 2\. Navigate to project folder:

# 

# ```

# cd your-repo-name

# ```

# 

# 3\. Install dependencies:

# 

# ```

# pip install -r requirements.txt

# ```

# 4\. Run the app:

# 

# ```
